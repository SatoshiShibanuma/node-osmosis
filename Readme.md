# Node Osmosis Web Scraper

## Project Overview
A robust and flexible web scraping library for Node.js.

## Prerequisites
- Node.js (>= 14.0.0)
- npm

## Installation
```bash
npm install
```

## Running Tests
```bash
npm test
```

## Development
- `npm run lint`: Run ESLint
- `npm run lint:fix`: Automatically fix linting issues

## Configuration
The project uses ESLint for code quality and style checking.

## Dependencies
- libxmljs-dom: XML parsing
- needle: HTTP requests
- nodeunit: Testing framework
